# Podlesny Twins — Credits

Портфолио-дискография студии сведения и мастеринга Podlesny Twins (Павел и Антон Подлесные). Витрина работ: треки и альбомы с обложками и воспроизведением, фильтры по жанрам и наградам. Цель — за секунды доказать масштаб («они это свели») и подтолкнуть к контакту.

- **Live:** https://credits.podlesnytwins.com
- **Главный сайт:** https://podlesnytwins.com (пункт меню «Работы» ведёт сюда)
- **Repo:** https://github.com/lesnycode/credits

## Стек и деплой

- Один статический файл `index.html` — весь HTML/CSS/JS инлайн. Без сборки, без фреймворков (нет React/Vite/Next/TS/Tailwind).
- Деплой: GitHub Actions → GitHub Pages при пуше в `main` (workflow «Deploy static site to Pages»). Кастомный домен в `CNAME`.
- `gh run watch <id>` — следить за деплоем.

## Архитектура index.html

- `.pf` — корневой контейнер, тёмная сцена (`#1a1a19`).
- `.pfnav` — шапка: лого `.pfbrand` (→ главный сайт) слева, `.navlinks` справа: `.pflink.pfworks` («Наши работы», активный красный, текущий раздел) + `.pflink` «Курс» (→ podlesnytwins.com), между ними hairline-разделитель (`border-left` на втором `.pflink`), оба uppercase в тон фильтрам. Стрелка `.ar` у «Курс» скрыта внутри `.navlinks`. **Важно:** класс `.pfmenu` уже занят дропдауном в футере (`#pfmenu`) — для шапки использовать `.navlinks`, не переиспользовать `.pfmenu`.
- `.hero` — тонкая полоса-осциллограф (~18px десктоп / 16px мобила), `<canvas id="waves">`, **без баннера-заголовка**. Фон прозрачный, canvas чистится `clearRect` (тёмной заливки-«блока» нет) — видна только линия (красная + приглушённая серая) по центру (`y=H/2`). Амплитуда масштабируется к высоте (`hs=H/100`), есть idle-baseline (линия «живёт» и реагирует на курсор без проигрывания, сильнее при play), уважает `prefers-reduced-motion`. Заголовок `<h1>Podlesny Twins Credits</h1>` сохранён как `.sr-only` (визуально скрыт, для SEO), сам hero `aria-hidden`.
- `.filters` — типографская строка фильтров (без pill-кнопок). `data-f` = жанр (ALL/HIPHOP/POP/INDIE/ROCK) или награда (GRAM/TOP1). Счётчики `.fc` считаются JS из `data-g`/`data-aw`.
- `.grid` / `.tile` — карточки треков и альбомов. Одиночный трек: `data-g` (жанр), `data-id` (Spotify ID), `onclick="play(this)"`. Альбом: `.tile.album`, `onclick="openAlbum(N)"`, данные в массиве `ALBUMS`.
- Порядок тайлов: сгруппированы по основному артисту (треки одного артиста идут подряд), коллабы — по первому артисту в строке («Dima Bilan» и «Дима Билан» = один). Сверху закреплены группы: PIZZA → Билан → MACAN&NAVAI → T-Fest → Андрей Катиков; далее остальные по первому появлению. Дублей тайлов быть не должно (дедуп по `data-id` / по альбому).
- Награды: атрибут `data-aw` на тайле (`GRAM`/`TOP1`) — управляет только фильтрацией (визуальных бейджей на карточках нет, убраны).
- Плеер: нижняя панель `#player`, использует **Spotify iFrame API** (`onSpotifyIframeApiReady` → `spotifyCtrl`). `play(el)` делает `loadUri`+`play` внутри клика; повторный клик по активной карточке = `togglePlay`. Иконка ▶ → пауза (две полоски `.pb`) по событию `playback_update`.
- Мобильный автоплей (iOS/Safari часто блокирует): флаг `wantPlay` + класс `.tile.retap` — пульс play-кнопки (`@keyframes retap`), подсказка «тапни ещё раз». В `playback_update`: если `wantPlay && isPaused` → `resume()` + `.retap`; при реальном старте `.retap` снимается. Логика чувствительная — менялась братом несколько раз, текущая версия «retap + wantPlay».
- Модалка альбома `#amodal`: `role="dialog"`, закрытие по Esc и клику по фону.

## Бренд / правила дизайна

- Регистр: **brand** (design IS the product). Характер: профи, мощь, премиальность.
- Анти-референс: дженерик-лендинг «от нейросети» (шаблонные pill-кнопки, дефолтные шрифты, одинаковые карточки-сетки, разноцветный glow). Если не отличить от тысячи AI-лендингов — провал.
- Один акцент: брендовый красный `#cf2c04` (идентичен главному сайту). Всё прочее — оттенки тёмного.
- Заголовочный шрифт `SaarSP` (заглавные, тяжёлый рубленый) — с Tilda CDN, тот же, что на главном сайте. Текст — Work Sans.
- WCAG AA: контраст ≥4.5:1, видимый фокус, тач-таргеты ≥44px.

## Контент / конвенции

- Обложки: `https://i.scdn.co/image/ab67616d0000b273<hash>` (640px). Хеш брать из Spotify oEmbed (`/oembed?url=...` → `thumbnail_url`, заменить размер на `ab67616d0000b273`).
- Жанр трека — атрибут `data-g`. Перенос жанра = смена `data-g`, счётчики пересчитываются сами.
- Множественное число треков — JS-хелпер `plt(n)` (трек/трека/треков). На бейджах альбомов писать полным словом, не «тр.».
- **Иноагенты:** карточки Morgenshtern помечены `<sup class="ia">*</sup>`, в футере сноска `.iadisc`. При добавлении контента иноагентов — тот же маркер + сноска.
