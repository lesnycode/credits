2. **Почему артисты разделены точкой с запятой (;)**

   **Причина:**
   - Spotify CSV экспорт (Artist Name(s)) использует `;` для коллабов: "PIZZA;NAVAI", "T-Fest;OG Buda", "MACAN;NAVAI" и т.д.
   - `build_all.py` брал `full = r.get("Artist Name(s)")` полностью (prim() только для группировки).
   - Это попадало в `<span class="art">PIZZA;NAVAI</span>`.
   - `build_who_mixed.py` парсил это как есть → трек-страницы и /track/ хаб тоже показывали ; 

   **Что сделали:**
   - Добавили `norm_art()`: `;` → `, ` в обоих генераторах.
   - Поправили текущий `works-site/index.html` (тайлы + alts + JSON).
   - Пересобрали `works-site/track/` (205 страниц) — теперь везде "PIZZA, NAVAI", "BATO, SALUKI" и т.д.
